# Vue.js 購物網站

## Demo
https://yangjiesu.github.io/PlayStation-custom/dist

## Intro
* 將後臺所建立的商品及購買系統利用 Ajax 將畫面呈現至前台客戶端

### 1. All Products
* 所有商品頁面
* 商品皆可於後台系統所建立

### 2. Single Product 
* 查看單一商品細節及訂購商品

### 3. Checkout 
* 購買清單及訂購人所需填的資料
* 使用 VeeValidate 進行資料驗證
